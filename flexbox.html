  • display:flex - used to display texts in boxes
  • flex:number - divide the boxes in ratio
  • order:number - order it
  • align-items: flex-start - starts from start and freespace is reduced
  • align-items: flex-end - starts from end and freespace is reduced
  • align-items: center - starts from center and freespace is reduced
  • flex-direction: column - display columnwise
  • align-items: start - aligns the box on start
  • align-items: end - aligns the box on end
  • justify-content:center - aligns the box on center
  • justify-content:space-between - freespace b/w boxes
